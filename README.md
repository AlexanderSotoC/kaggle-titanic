# Titanic: Machine Learning from Disaster

This is an IPython Notebook for the Kaggle competition, Titanic: Machine Learning from Disaster. The goal of this repository is to provide an example for the prediction of the survival or not of the passengers of Titanic.

### Some libraries used:
* [NumPy](http://www.numpy.org/)
* [Pandas](http://pandas.pydata.org/)
* [Seaborn](https://seaborn.pydata.org/)
* [Matplotlib](http://matplotlib.org/)
* [Scikit-Learn](http://scikit-learn.org/stable/)
* [XGBoost](https://xgboost.readthedocs.io/en/latest/#)
